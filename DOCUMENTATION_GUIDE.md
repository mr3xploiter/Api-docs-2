# Media Upload API - Documentation Guide

## Overview

This documentation portal contains comprehensive guides for the Media Upload API - a production-ready, self-hosted media upload service built with Express.js.

## Documentation Structure

### Main Portal
- **index.html** - Main documentation portal with interactive navigation, nested sidebar, and table of contents

### Getting Started Guides
- **Quick Start** (in index.html) - Get up and running in 5 minutes
- **Installation** (in index.html) - Multiple installation methods
- **API Endpoints** (in index.html) - Overview of all available endpoints

### API Reference
- **api-detailed.html** - Complete detailed reference for each API endpoint with:
  - Request/response examples
  - Parameter definitions
  - Error codes
  - Size limits
  - Format specifications

### Code Examples
- **code-examples.html** - Copy-paste ready examples covering:
  - Basic image upload
  - Upload with progress tracking
  - Drag and drop upload
  - Multiple file uploads
  - Video uploads
  - File operations (get, delete, list)
  - Storage statistics
